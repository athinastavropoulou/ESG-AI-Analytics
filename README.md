# AI-Driven Business Analytics for ESG Performance 📊🌱

## Project Overview
This project explores the intersection of **Artificial Intelligence (AI) maturity** and **Environmental, Social, and Governance (ESG)** performance. By analyzing 27 global corporations, the research identifies how data-driven decision-making and predictive analytics correlate with sustainability outcomes.

This repository contains the technical workflow, data cleaning process, and advanced clustering analysis developed as part of my undergraduate thesis at the **Athens University of Economics and Business (AUEB)**.

## Key Insights
* **Correlation Found:** A **0.53 positive correlation** between AI Maturity and Environmental Scores, proving that digital transformation is a key driver for decarbonization.
* **The Resource Paradox:** Higher revenue/investment does not automatically guarantee better ESG performance; strategic data alignment is more critical than capital.
* **Clustering:** Using K-Means, companies were segmented into three strategic groups: *ESG-AI Leaders*, *Tech Disruptors*, and *Sustainability Laggards*.

## Technical Stack
* **Language:** Python 3.x
* **Libraries:** * `Pandas` & `NumPy` for data manipulation.
  * `Matplotlib` & `Seaborn` for advanced data visualization.
  * `Scikit-learn` for K-Means Clustering and Data Scaling.
* **Environment:** Google Colab / Jupyter Notebooks.

## Analytical Workflow
1. **Data Preprocessing:** Handling missing values, correcting scaling errors (e.g., ESG scores), and treating outliers in Tech Investment data.
2. **Exploratory Data Analysis (EDA):** Visualizing distributions across industries and identifying performance benchmarks.
3. **Statistical Analysis:** Pearson correlation matrix to identify relationships between AI capabilities and ESG pillars (E, S, G).
4. **Machine Learning:** Implementing **K-Means Clustering** to segment the market based on digital and sustainability maturity.
